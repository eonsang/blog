# 실무에서 쓰는 TypeScript 팁 5가지

TypeScript를 사용하면서 알아두면 좋은 패턴들을 소개합니다.

## 1. 타입 좁히기 (Type Narrowing)

조건문으로 타입을 좁혀서 안전하게 사용할 수 있습니다.

```typescript
function processValue(value: string | number) {
  if (typeof value === "string") {
    return value.toUpperCase();
  }
  return value.toFixed(2);
}
```

## 2. Discriminated Union

`type` 필드로 유니온을 구별하는 패턴입니다.

```typescript
type Result =
  | { status: "success"; data: string }
  | { status: "error"; message: string };

function handle(result: Result) {
  if (result.status === "success") {
    console.log(result.data);
  } else {
    console.error(result.message);
  }
}
```

## 3. satisfies 키워드

타입 추론을 유지하면서 타입 검증을 할 수 있습니다.

```typescript
const config = {
  port: 3000,
  host: "localhost",
} satisfies Record<string, string | number>;
```

## 4. 제네릭 활용

재사용 가능한 타입 안전한 함수를 만들 수 있습니다.

```typescript
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key];
}
```

## 5. Utility Types

`Partial`, `Pick`, `Omit` 등 내장 유틸리티 타입을 적극 활용하세요.

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

type UpdateUser = Partial<Pick<User, "name" | "email">>;
```

이 패턴들을 잘 활용하면 타입 안전성과 개발 생산성을 동시에 높일 수 있습니다.
