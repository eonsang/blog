# Hello World

블로그에 오신 것을 환영합니다. 이 블로그는 **Next.js**와 **Velite**를 사용하여 만들어졌습니다.

## 기능

이 블로그는 다음과 같은 기능을 지원합니다:

- MDX 콘텐츠 작성
- 코드 구문 강조
- 목차 자동 생성
- 태그 기반 분류
- 시리즈 포스트
- RSS 피드
- 다크 모드

## 코드 예시

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
